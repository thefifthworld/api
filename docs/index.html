The Fifth World presents an open source shared universe set in a neotribal, ecotopian future.

The Fifth World website provides a hub for a community of writers, artists, gamers, and others who create and use this setting, adding their stories, characters, and work to the world.

The Fifth World API provides a RESTful interface to create, edit, and modify the database where we store our collaborative work.

Example: The Fifth World Website

The Fifth World website uses the API, making it an excellent example of how you can use it for your own purposes. Throughout this documentation, we'll use the website as an example of what the API can do.
