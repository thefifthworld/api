Edit member

This endpoint allows you to make changes to a member’s account information. You must have permission to do so, so you must authenticate as the member you want to change, or as a member who has administrative privileges. The endpoint updates only the fields you provide, leaving all others unchanged.

Request

HTTP Headers
HTTP Header Value
Authorization You must provide a valid JSON Web Token that authenticates you either as an administrator or as the member you wish to edit.
Path Parameters
Path parameter Value
id The unique ID number of the member you’d like to edit.
Member object
Property Value type Value
name string The member’s name.
email string The member’s email address.
password string The member’s password.
bio string A short biography or description for the member.
facebook string The member’s Facebook URL.
twitter string The member’s Twitter URL.
github string The member’s GitHub URL.
patreon string The member’s Patreon URL.
web string The URL of the member’s website.

Examples

Response

If the endpoint successfully updated the member account, it returns a 200 HTTP status with a JSON object representing the updated account information. If the endpoint could not authenticate you as either the member to edit or an administrator, it responds with a 401 HTTP status.
