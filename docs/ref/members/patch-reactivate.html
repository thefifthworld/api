Reactivate member

This endpoint reactivates a previously deactivated member. Only administrators can use it.

Request

HTTP Headers
HTTP Header Value
Authorization You must provide a valid JSON Web Token to authenticate yourself as an administrator to use this endpoint.
Path Parameters
Path parameter Value
id The unique ID number of the member you’d like to reactivate.

Examples

Response

If you authenticate with an account with administrative privileges, the endpoint will reactivate the account requested and respond with a 200 HTTP status and a JSON object representing the account’s updated information.

If you failed to authenticate, or authenticated with an account that does not have administrative privileges, the endpoint responds with a 401 HTTP status.
